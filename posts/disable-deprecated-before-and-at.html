29 Mar 2021

-DKF_DISABLE_DEPRECATED_BEFORE_AND_AT=0x054F00 

0x054F00
0x05    0x4F   0x00

In terminal: 

$ printf '%d %d %d\n' 0x05 0x4F 0x00
5 79 0

Using std::cout: 

std::cout << std::dec << 0x05 << " "
                      << 0x4F << " "
                      << 0x00 << std::endl;

Outputs:
5 79 0

And: 

std::cout << std::hex << QT_VERSION_CHECK(5, 79, 0) <<  std::endl;

Outputs:
54f00

                   54f00
Hexacdecimal: 54    4f   00
Decimal:      5     79   0
